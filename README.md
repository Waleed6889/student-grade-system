# Student Grade System using JavaScript

## 📚 Project Overview

This is a beginner JavaScript project that checks a student's result based on their name and roll number.

The user enters:

* Student Name
* Roll Number

If the information matches the stored data, the program displays:

* Mathematics Marks
* Science Marks
* English Marks
* Total Marks
* Percentage

If the information is incorrect, an error message is displayed.

## 💻 Technologies Used

* HTML5
* CSS3
* JavaScript

## ✨ Features

* Student name verification
* Roll number verification
* Case-insensitive name matching
* Percentage calculation
* Error handling for invalid data
* Responsive and clean user interface

## 🚀 Future Improvements

* Add grade calculation (A+, A, B, etc.)
* Store student data using arrays and objects
* Search multiple students dynamically
* Connect with a database
